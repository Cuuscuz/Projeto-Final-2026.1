Gerenciador de Tarefas

Sobre o projeto

Projeto desenvolvido por Alisson Moreira de Araújo e Yuri Sousa Pereira como trabalho final da disciplina.

Motivação

A ideia surgiu a partir das atividades de cadastro de usuário trabalhadas em sala pelo professor. Aproveitando os mesmos conteúdos e princípios aplicados nessas atividades, o grupo decidiu adaptar o conceito de "cadastro de usuários" para a criação de um Gerenciador de Tarefas, atingindo o objetivo final da disciplina.

Objetivo

Organizar tarefas pessoais — realizadas ou a realizar — substituindo o controle manual em papel por uma ferramenta digital simples e prática.

Funcionalidades

Criar tarefas: adicionar novas tarefas a serem realizadas.
Editar tarefas: modificar tarefas já cadastradas.
Concluir tarefas: marcar tarefas como "Concluída".
Excluir tarefas: remover tarefas existentes.
Tarefas pendentes: visualizar a lista de tarefas ainda não concluídas.
Tarefas concluídas: visualizar a lista de tarefas já finalizadas.
